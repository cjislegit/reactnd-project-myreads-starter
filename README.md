# MyReads Project

A web application that allows you to easily categorize books you have read, want to read or are currently reading.

## Installation

```
Clone or fork repo
cd into reactnd-project-myreads-starter
run npm install
```

## Usage

```
cd into reactnd-project-myreads-starter
run npm start
```
